# Assets Directory

Place your logo and icon files here:

## Logo
- **File**: `src/renderer/assets/logo.png` (or `.jpg`, `.svg`)
- **Usage**: Used in the app UI (HomeView, Sidebar)
- **Recommended size**: 512x512px or larger (will be scaled automatically)

## App Icon
- **Windows**: `assets/icon.ico` (256x256px recommended)
- **macOS**: `assets/icon.icns` (512x512px recommended)
- **Linux**: `assets/icon.png` (512x512px recommended)
- **Fallback**: `assets/icon.png` (used if platform-specific icon not found)

The app will automatically use the appropriate icon for your platform.
